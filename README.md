### [pitchchanger](./pitchchanger.lua) for mpv

Control audio pitch by 0.01.
<kbd>Ctrl+]</kbd> increases,
<kbd>Ctrl+[</kbd> decreases 


## Installation

put them in the Scripts config

and add this line to mpv.conf

<kbd>af=@rb:rubberband</kbd>
